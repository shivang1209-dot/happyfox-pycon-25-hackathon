# **How to Use**

1. **Add your OpenAI API key to `.env` file:**
   ```
   OPENAI_API_KEY=your-api-key-here
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the system:**
   ```bash
   python main.py
   ```

4. **Check results in `output_result.json`**
